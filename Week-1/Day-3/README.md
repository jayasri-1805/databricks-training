# SQL Window Functions and CTE Assignment

## 📌 What I Practiced

This repository contains SQL practice queries based on Window Functions and Common Table Expressions (CTEs) using an Employee-Customer-Orders database schema as part of my DBMS and SQL learning journey.

The practice mainly focused on:

* Ranking and analytical window functions
* Running totals and cumulative calculations
* LAG() and LEAD() functions
* Partitioning data using PARTITION BY
* Aggregate analysis with window functions
* Common Table Expressions (CTEs)
* Recursive CTEs
* Combining CTEs with window functions
* Solving advanced SQL analytical problems
